# Tweeter Project

Tweeter is a simple, single-page Twitter clone. It uses HTML, CSS, JS, jQuery and AJAX front-end, with Node.

## Dependencies
- body-parser
- chance
- express
- md5

## Getting Started
- Install all dependencies (using the npm install command).
- User 'npm start' command to start server
